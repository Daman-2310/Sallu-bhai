# 🚨 SALLU BHAI · Footpath Survival Portal 💀

A cinematic, single-page 3D web experience. Sallu Bhai is driving — touch the lucky
charm, survive the SUV on the footpath, and get savagely roasted.

Built with **Three.js** (chrome *firoza* bracelet, deep-space nebula, real bloom),
**GSAP** cinematic camera + screen-shake, a **WebAudio** engine-rev / tyre-screech /
impact synth, phone haptics, gyro tilt, and a matrix-style roast decoder.

## ▶ Run it
Just open `index.html` in any modern browser. No build step, no dependencies to
install — everything loads from CDN.

## ✨ Features
- Deep-space starfield + drifting nebula clouds + radial dark vignette
- Floating chrome turquoise bracelet with real environment reflections + neon bloom
- Hover / touch / gyro reactive 3D
- "Footpath Emergency" panic flow: SUV with blinding headlights rushes the screen
- Crimson **Boys** / hot-pink **Girls** cyberpunk choice buttons
- Random savage roast engine with matrix-decode animation
- Survival-streak counter, share button, "Run Away Safely" reset
